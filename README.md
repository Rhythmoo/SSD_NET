# SSD_NET
---
训练能识别车灯与祈年殿的SSD模型，模型训练过程参考源码：https://github.com/balancap/SSD-Tensorflow ，在此基础上用自己制作的数据集训练，以达到能识别
车灯与祈年殿的目的。此两类目标的选择主要是为了比对不同尺寸大小的物体的识别率。

#### 运行环境
python3.6 ,tensorfolw1.4, flask

#### 运行界面
执行 python web_ssd.py 在本地运行，路径需要根据本地配置
